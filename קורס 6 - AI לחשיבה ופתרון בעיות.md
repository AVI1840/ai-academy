# קורס 6: AI לחשיבה ופתרון בעיות מורכבות
## מבוסס על: Features of Claude — Extended Thinking + Reasoning (Anthropic + OpenAI)

---

### למי המדריך מיועד?
למנהלים, מקבלי החלטות ועובדים שמתמודדים עם בעיות מורכבות שדורשות ניתוח מעמיק.

---

### חלק א׳: AI כשותף לחשיבה (10 דקות)

**AI לא רק כותב — הוא גם חושב.** כשנותנים לו בעיה מורכבת, הוא יכול:
- לפרק אותה לחלקים
- לבחון אותה מזוויות שונות
- להציע פתרונות שלא חשבת עליהם
- לזהות סיכונים ומכשולים

**מתי להשתמש ב-AI לחשיבה?**

| מצב | דוגמה מביטוח לאומי |
|------|-------------------|
| תכנון פרויקט | "איך ליישם מערכת תורים דיגיטלית בסניפים?" |
| פתרון בעיה | "למה זמני הטיפול בתביעות עלו ב-30%?" |
| קבלת החלטה | "האם כדאי לאחד שני סניפים? מה היתרונות והחסרונות?" |
| שיפור תהליך | "איך לקצר את תהליך הגשת תביעת נכות?" |

---

### חלק ב׳: טכניקות חשיבה עם AI (15 דקות)

**טכניקה 1: ניתוח SWOT**

```
בצע ניתוח SWOT למהלך הבא: הטמעת צ׳אטבוט AI לשירות תושבים בביטוח לאומי.
התייחס ל: חוזקות, חולשות, הזדמנויות, איומים.
כלול שיקולים של: פרטיות, נגישות, עלות, שביעות רצון תושבים.
```

**טכניקה 2: חשיבה מובנית צעד-אחר-צעד**
```
אני צריך לשפר את תהליך הטיפול בתביעות אבטלה.
חשוב על זה צעד אחר צעד:
1. מה התהליך הנוכחי?
2. איפה צווארי הבקבוק?
3. מה אפשר לייעל עם טכנולוגיה?
4. מה אפשר לייעל בלי טכנולוגיה?
5. מה סדר העדיפויות?
6. מה הסיכונים בכל שינוי?
```

**טכניקה 3: "עורך דין של השטן"**
```
אני מתכנן [תאר את התוכנית].
שחק תפקיד של מבקר ביקורתי:
- מה יכול להשתבש?
- מה אני לא רואה?
- מה ההתנגדויות הצפויות?
- מה התרחיש הגרוע ביותר?
```

**טכניקה 4: סימולציית תפקידים**
```
אני מתכונן לפגישה עם [מנהל בכיר / נציג אוצר / ועד עובדים].
שחק את התפקיד שלהם ושאל אותי שאלות קשות שהם עשויים לשאול.
הנושא: [תאר]
```

**טכניקה 5: ניתוח שורש הבעיה (5 Whys)**
```
הבעיה: תושבים מתלוננים על זמני המתנה ארוכים בסניף.
בצע ניתוח "5 למה" — שאל "למה?" 5 פעמים כדי להגיע לשורש הבעיה.
לכל שלב, הצע פתרון אפשרי.
```

### חלק ג׳: פרומפטים מוכנים לחשיבה ניהולית

**הכנת נייר עמדה:**
> "כתוב נייר עמדה בן עמוד אחד בנושא [X]. מבנה: רקע, מצב קיים, חלופות, המלצה, צעדים הבאים"

**הכנה לישיבה:**
> "אני הולך לישיבה בנושא [X]. הכן לי: 5 נקודות מפתח להעלות, 3 שאלות לשאול, 2 סיכונים לציין"

**סיעור מוחות:**
> "אני צריך 10 רעיונות יצירתיים לשיפור שירות התושבים בביטוח לאומי. חשוב מחוץ לקופסה. כלול רעיונות טכנולוגיים ולא-טכנולוגיים"

**טכניקה 2: חשיבה מזוויות שונות**
```
אני שוקל לשנות את תהליך קליטת תביעות נכות לתהליך דיגיטלי מלא.
נתח את הרעיון מ-4 זוויות:
1. זווית התושב — מה הוא ירוויח/יפסיד?
2. זווית העובד — איך זה ישפיע על העבודה שלו?
3. זווית הארגון — עלויות, יעילות, סיכונים
4. זווית משפטית — האם יש מגבלות רגולטוריות?
```

**טכניקה 3: "עורך דין של השטן"**
```
אני מתכנן [תאר תוכנית].
שחק תפקיד של מבקר ביקורתי:
- מצא 5 חולשות בתוכנית
- לכל חולשה הצע פתרון
- דרג את הסיכונים מגבוה לנמוך
```

**טכניקה 4: תכנון לאחור**
```
המטרה: להפחית את זמן הטיפול הממוצע בתביעות ב-25% תוך שנה.
בנה תוכנית עבודה לאחור:
- מה צריך לקרות בחודש 12?
- מה צריך לקרות בחודש 6?
- מה צריך לקרות בחודש 1?
- מה הצעד הראשון מחר בבוקר?
```
