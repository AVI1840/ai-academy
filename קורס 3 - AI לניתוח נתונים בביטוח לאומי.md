# קורס 3: AI לניתוח נתונים בביטוח לאומי
## מבוסס על: Build with the Claude API — Data Analysis (Anthropic + OpenAI)

---

### למי המדריך מיועד?
לעובדים שמתעסקים עם נתונים, דוחות וסטטיסטיקות — מנהלי סניפים, אנליסטים, עובדי תכנון ומחקר.

---

### חלק א׳: מה AI יכול לעשות עם נתונים? (10 דקות)

**AI מצטיין ב:**
- זיהוי מגמות בנתונים גדולים ("כמה תביעות נכות עלו ב-2024?")
- סיכום טבלאות ודוחות ארוכים
- יצירת גרפים והמחשות מנתונים
- זיהוי חריגות ("איזה סניף חורג מהממוצע?")
- כתיבת נוסחאות Excel ושאילתות SQL

**דוגמאות מביטוח לאומי:**

| משימה | פרומפט לדוגמה |
|--------|---------------|
| ניתוח עומסים | "הנה נתוני פניות לפי חודש. זהה את החודשים העמוסים ביותר והצע הסבר" |
| זיהוי מגמות | "נתח את הטבלה הבאה וזהה מגמות בתביעות נכות ב-3 השנים האחרונות" |
| דוח ניהולי | "על בסיס הנתונים, כתוב פסקת סיכום ניהולית ל-3 שורות" |
| נוסחאות | "כתוב נוסחת Excel שמחשבת אחוז שינוי בין חודשים" |

---

### חלק ב׳: איך לעבוד עם נתונים ב-AI — צעד אחר צעד (15 דקות)

**שלב 1: הכן את הנתונים**
- העתק טבלה מ-Excel (ללא מידע מזהה!)
- או תאר את הנתונים במילים: "יש לי טבלה עם עמודות: חודש, מספר תביעות, זמן טיפול ממוצע"

**שלב 2: בקש ניתוח ממוקד**
```
הנה נתוני תביעות לפי סניף ברבעון האחרון:
[הדבק טבלה]

אנא:
1. זהה את 3 הסניפים עם זמן הטיפול הארוך ביותר
2. חשב ממוצע ארצי
3. הצע 2 סיבות אפשריות לפערים
4. הצג את התוצאות בטבלה מסודרת
```

**שלב 3: בקש המחשה**
> "צור קוד Python ליצירת גרף עמודות שמציג את מספר התביעות לפי סניף"

**שלב 4: בקש סיכום ניהולי**
> "כתוב פסקת סיכום של 5 שורות למנהל, עם הממצאים העיקריים וההמלצה"

---

### חלק ג׳: 5 פרומפטים מוכנים לניתוח נתונים

**1. ניתוח טבלה:**
> "נתח את הטבלה הבאה. ציין: ממוצע, חציון, ערך מקסימלי ומינימלי, ומגמה כללית. [הדבק טבלה]"

**2. זיהוי חריגות:**
> "בדוק את הנתונים הבאים וזהה ערכים חריגים שחורגים מ-2 סטיות תקן מהממוצע. הסבר למה הם חריגים. [הדבק]"

**3. השוואת תקופות:**
> "השווה את הנתונים של רבעון 1 לרבעון 2. מה השתנה? מה ההשלכות? הצג בטבלת השוואה. [הדבק]"

**4. יצירת דוח:**
> "על בסיס הנתונים, כתוב דוח קצר (עמוד אחד) שכולל: רקע, ממצאים עיקריים, מסקנות, המלצות. [הדבק]"

**5. נוסחאות Excel:**
> "אני צריך נוסחת Excel שעושה [תאר]. העמודות שלי הן: A=תאריך, B=מספר תביעות, C=זמן טיפול. תן לי את הנוסחה עם הסבר"

---

### ⚠️ זהירות עם נתונים!
- **לעולם** אל תעלה נתונים עם פרטים מזהים (ת.ז., שמות, כתובות) לכלי AI חיצוני
- עבוד עם נתונים מצרפיים (סיכומים, ממוצעים) או נתונים אנונימיים
- לניתוח נתונים רגישים — השתמש רק בכלים מאושרים ע"י הארגון
